# toft

#### 介绍
使用hive的游戏框架，类似使用skynet的项目

#### 软件架构
按通常做法分为master\gate\global\game\login, 其中gate\game\login可以多开，只需要修改少量代码


#### 安装教程

### windows
1.  .\bat\build.bat
2.  .\bat\config.bat
3.  .\bat\proto.bat

### linxu & macosx
1.  ./shell/build.sh
2.  ./shell/config.sh test
3.  ./shell/proto.sh

#### 使用说明

### windows
1.  .\bat\start.bat

### linxu & macosx
1.  ./shell/service.sh
2.  ./shell/service.sh start
3.  ./shell/service.sh kill
